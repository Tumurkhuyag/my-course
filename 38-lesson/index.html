Lesson 38

git үндсэн командууд, git init, status, add, commit, checkout, push, pull командууд

  • git init - Тухайн төслийн хавтас дотор хоосон git repository үүсгэж өгнө
  • git config -global user.name "NAME": Энэ git систем дээр ажиллаж буй хүний нэр
  • git config -global user.email "email@address.com": Энэ git систем дээр ажиллаж буй хүний имэйл

    • Үндсэн командууд

  • git status одоо төслийн файлууд ямар байдалтай байгааг харуулна
  • git add "файлын нэр" файлын өөрчлөлтийг repository -д хадгалахаар сонгоно
  • git add -A бүх файлын өөрчлөлтийг repository -д хадгалахаар сонгоно
  • git commit -m "Тайлбар" сонгосон файлыг repository -д бичнэ
  • git commit -am "Тайлбар" файлын өөрчлөлтийг хадгалах болон сонгосон файлыг repository -д бичих үйлдлийг хамтад нь хийнэ
  • git checkout repository дотроос файлыг буцаан гаргаж авна
  • git push origin master local repository -оос remote repository -руу upload хийнэ
  • git pull origin master remote repository -оос local repository -руу файлынбуцаан татаж авна
  • git log өмнөх commit -уудаа харна
  • git branch local repository дээр хэдэн branch байгааг харна
