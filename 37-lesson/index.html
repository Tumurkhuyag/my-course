Lesson 37

git гэж юу вэ? gitbash суулгах

Git нь файлын өөрчлөлтийн түүхийг удирдах систем юм. Дараах зүйлсийг хийдэг. Үүнд:

  • Файлын өөрчлөлтийн түүх (change history)
  • Файл дээрх хамтын ажиллагааг боломжтой болгох (collaboration)
  • Нэг файлын өөр өөр хувилбаруудыг үүсгэж тусд нь ажиллах боломж (feature branches)
  • 2005 онд Linus Trowald зохиосон (Linux -ийн бүтээгч)

    • Зарим нэг зарчмууд

  • git checkout командыг ашиглаж өөртөө файлыг хуулж авна
  • git commit командыг ашиглаж үндсэн файлруу хуулбаруудыг оруулна
  • git merge командыг ашиглаж хуулбаруудыг нэгтгэнэ
