Хичээл 42

Эвент гэж юу вэ?
Эвент нь веб хуудас дээр ямар нэг үйлдэл хийгдлээ гэж броузераас Javascript кодонд мэдэгддэг процесс юм

Эвент боловсруулагч гэж юу вэ?
Эвент боловсруулагч (event listener) нь уг эвентийг хүлээж аваад хариу үйлдэл хийдэг Javascript дээрх бидний бичсэн функц юм

Message queue
Веб хуудас дээр үүссэн эвентүүдийн боловсруулагдахаа хүлээж оочирлодог газар юм

getElementById
DOM -оор дамжуулан HTML доторх обьектийг ID -аар дамжуулан олдог функц

Callback функц
А функцийн аргументруу Б функцийг дамжуулъя. А функцийн дотор дуудаж ажиллуулж буй Б функц нь Callback функц болно.

Anonymous функц
Функцийн аргументруу нэргүй функц шууд бичиж дамжуулж болдог. Нэргүй функцийг anonymous функц гэж гэдэг.
