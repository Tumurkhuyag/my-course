Lesson 28

Javascript хөшигний цаана - Javascript engine, Глобаль объектийн тухай

  1. Javascript код ажиллахад заавал нэг глобал обьект үүсэн байх шаардлагатай.
  2. Энэ объект нь Javascript кодод ажиллах орчных нь талаар мэдээлэл өгдөг.
  3. Бүх кодоос ашиглаж болох өгөгдөл функцээр хангана.
  4. Энэ объект доторх хувьсагч функцийг дуудахад объектийн нэрийг ашиглах албагүй байдаг.
  5. Энэ объект дотор зарлагдсан хувьсагч, функүүдийг глобал хувьсагч, функц гэдэг.

globalThis объект дотор primitive өгөгдөл бүхий хувьсагчийг жижиг үсгээр нэршилжүүлдэг
