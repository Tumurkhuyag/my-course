Төсвийн мэдээллийг дэлгэц дээр харуулах

Төслийн хэсэг

appController модул дотор getBalance(); функцийг financeController -оос дуудаж balance хувьсагч дотор хадгалах

uiController() модулын DOMstrings обьект дотор balance -ийн утгуудыг харуулах class -уудыг нэмэх

uiController() модул дотор showBalance функ үүсгэж HTML дээр balance -ын утгуудыг харуулах програмчлалыг бичих

uiController дотор showBalance() функцийг дуудаж getBalance -ын утгуудыг дамжуулж дэлгэц дээр утгуудыг харуулах

хувь харуулдаг хэсэг 0 -оос их үед % тэмдэгтэй болгох
