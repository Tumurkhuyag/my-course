let, const ашиглан хувьсагч зарлах

Суралцсан

{const birthDate = 1995} гэж зарласан тохиолдолд birthDate -д өгсөн 1995 утгыг дараа нь огт өөрчлөх боломжгүй болно

Дараа нь огт өөрчлөхгүй утгатай хувьсагчийг const ... гэж зарладаг

{let mouse = 200} гэж зарласан тохиолдолд mouse гэсэн хувьсагчид оногдсон утга зөвхөн block scope дотор буюу {} энэ хоёр хаалтын хооронд л ажиллана

const ч мөн адил зөвхөн block scope дотор буюу {} энэ хоёр хаалтын хооронд л ажиллана

Эхлээд бүх хувьсагчаа const -оор зарлаж байгаад зарим хувьсагчийг let гэж зарлах шаардлагатайг мэдээд let -ээр өөрчилж зарлаж болно

Эсрэгээр нь бүх хувьсагчуудаа let -ээр зарлаж байгаад өөрчлөгдөж болохгүй хувьсагчуудыг const -оор өөрчилж зарлаж болно
