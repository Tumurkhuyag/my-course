Lesson 111

Веб апп #3 - Хоолны жор, веб сайтын webpack, babel тохиргоог гүйцэтгэж шалгах

Төслийн хэсэг

Фолдер зохион байгуулалт
dist, src фолдерууд үүсгэх
src -руу index.html, dist -рүү css, img -г хуулах

package.json файл үүсгэх
npm init -y

webpack & webpack-cli devDependencies байдлаар суулгах
npm install webpack, webpack-cli --save-dev

webpack -ын тохиргооны файлыг үүсгэх
webpack.config.js

HTML Webpack Plugin суулгах
npm i --save-dev html-webpack-plugin
webpack.config.js дотор зарлаж, plugins дотор тохируулах

package.json дотор command script бичсэн, mode -тэй нь цуг
"dev": "webpack --mode development",
"build": "webpack --mode production"

Dev server суулгаж, тохиргоог нь хийх
npm install --save-dev webpack-dev-server
webpack.config.js => devServer: { static: './dist', },

Өмнө нь dist дотор үүссэн хэрэггүй файлуудыг цэвэрлэдэг болголоо
{output: /clean: true}

src файл -д хаана алдаа гарсныг харах боломжтой болголоо
devtool: "inline-source-map",

babel суулгах
npm install --save-dev @babel/core @babel/cli @babel/preset-env
npm install --save @babel/polyfill

babel loader суулгах
npm install -D babel-loader @babel/core @babel/preset-env webpack
