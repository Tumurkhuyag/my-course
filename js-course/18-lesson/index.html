Массивтай танилц, pop, push, shift, unshift, length

var proLang = ["c#", "java", "python", "javascript"];
proLang[1]: Массивийн өгөгдлийг байрлаж буй дугаараар нь дуудах
proLang.length: Массив доторх өгөгдлүүдийн тоог гаргах
proLang.unshift("dart"): Массивийн урд талаас нь өгөгдөл нэмэх
proLang.shift(): Массивийн урд талын өгөгдлийг хасах
proLang.push("swift"): Массивийн хойноос өгөгдөл нэмэх
proLang.pop(): Массивийн хамгийн сүүлд байрлаж буй өгөгдлийг дуудах
proLang.indexOf("javascript"): Өгөгдлийн утгаар массив дотроос хайлт хийж proLang[] index дугаарыг олох
